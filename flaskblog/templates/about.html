About

Welcome to Flask Blog App! This is a simple blogging platform built using the Flask framework. It allows users to create, read, update, and delete blog posts.

Flask Blog App was created by Uttam Patel as a personal project to learn and practice web development skills. It is open source and available on GitHub at Uttampatel1.

Technologies Used

  • Python 3.9
  • Flask 2.1.0
  • SQLite 3
  • HTML 5
  • CSS 3
  • Bootstrap 5.1.0

Features

  • User authentication
  • Create, read, update, and delete blog posts
  • Responsive design
  • Search functionality
  • Pagination

Getting Started

To run Flask Blog App on your local machine, follow these steps:

  1. Clone the repository: git clone [GitHub repository link]
  2. Navigate to the project directory: cd flask-blog-app
  3. Create a virtual environment: python3 -m venv venv
  4. Activate the virtual environment: source venv/bin/activate
  5. Install the required packages: pip install -r requirements.txt
  6. Create the database: flask init-db
  7. Run the app: flask run
  8. Open your web browser and go to http://localhost:5000
